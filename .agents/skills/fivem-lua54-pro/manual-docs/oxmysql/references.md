# oxmysql Manual Index

Open only the file that matches the database operation being implemented or reviewed.

- [query.md](./query.md) - Select multiple rows or run a general parameterized query.
- [single.md](./single.md) - Select exactly one row or the first matching row.
- [scalar.md](./scalar.md) - Select one value such as a count, id, or aggregate.
- [insert.md](./insert.md) - Insert a row and return the inserted id.
- [update.md](./update.md) - Update or delete rows and return the affected row count.
- [prepare.md](./prepare.md) - Reuse prepared SQL and return shape-dependent results.
- [rawExecute.md](./rawExecute.md) - Execute prepared SQL while always returning row arrays for selects.
- [transaction.md](./transaction.md) - Group multiple writes that must commit or roll back together.
